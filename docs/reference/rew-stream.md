# rew stream

Print arguments as lines

## Usage

```
rew stream [OPTIONS] [VALUES]...

```

## Arguments

<dl>
<dt><code>[VALUES]...</code></dt>
<dd>
Values to print
</dd>
</dl>

## Options

<dl>

<dt><code>-h, --help</code></dt>
<dd>
Print help (see a summary with '-h')
</dd>
</dl>
