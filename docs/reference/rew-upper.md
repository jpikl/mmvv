# rew upper

Convert characters to uppercase

## Usage

```
rew upper [OPTIONS]
```

## Options

<dl>

<dt><code>-h, --help</code></dt>
<dd>

Print help (see a summary with '-h')
</dd>
</dl>

## Global options

See [rew reference](rew.md#global-options) for list of additional global options.

## Examples

Convert characters to uppercase

```sh
> echo 'Hello World!' | rew upper
HELLO WORLD!
```
